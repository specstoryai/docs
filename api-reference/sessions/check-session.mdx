---
title: "Check Session"
api: "HEAD /api/v1/projects/{projectId}/sessions/{sessionId}"
description: "Check if a session exists and get metadata without retrieving content"
---

## Request

<ParamField path="projectId" type="string" required>
  The unique identifier of the project
</ParamField>

<ParamField path="sessionId" type="string" required>
  The unique identifier of the session
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="If-None-Match" type="string">
  ETag value for conditional requests
</ParamField>

## Response

This endpoint returns only HTTP status codes and headers, no response body.

### Response Headers

| Header | Description |
|--------|-------------|
| `ETag` | Entity tag for the session version |
| `Last-Modified` | Last modification timestamp |
| `Content-Length` | Size of the session content |

### Status Codes

| Status | Description |
|--------|-------------|
| `200` | Session exists |
| `304` | Not Modified (if ETag matches) |
| `401` | Unauthorized |
| `404` | Session not found |

import requests
