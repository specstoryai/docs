---
title: "Delete Session"
api: "DELETE /api/v1/projects/{projectId}/sessions/{sessionId}"
description: "Permanently delete a session and its content"
---

<Warning>
  This action cannot be undone. The session and all its content will be permanently deleted.
</Warning>

## Request

<ParamField path="projectId" type="string" required>
  The unique identifier of the project
</ParamField>

<ParamField path="sessionId" type="string" required>
  The unique identifier of the session to delete
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="success" type="boolean">
      Confirmation of successful deletion
    </ResponseField>
  </Expandable>
</ResponseField>

```bash cURL
curl -X DELETE \

```
